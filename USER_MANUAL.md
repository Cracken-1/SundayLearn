## Getting Started

### Accessing SundayLearn
1. Open your web browser
2. Navigate to your SundayLearn website URL
3. Browse the public content or click "Admin Login" to access the dashboard

### First Time Setup
1. **Admin Account**: Contact your system administrator for login credentials
2. **Password Change**: You may be required to change your password on first login
3. **Profile Setup**: Complete your profile information in Account Settings

## Public Website

### Home Page
- **Featured Content**: View highlighted lessons and resources
- **Quick Navigation**: Access main sections (Lessons, Resources, Blog)
- **Newsletter Signup**: Subscribe to updates and new content notifications

### Browsing Lessons
1. **Navigate to Lessons**: Click "Lessons" in the main menu
2. **Filter Options**:
   - Age Group: 3-5, 6-8, 9-12, Teen, Adult
   - Topic: Biblical stories, holidays, values
   - Search: Use keywords to find specific content
3. **View Lesson**: Click on any lesson card to view full details
4. **Download Resources**: Access worksheets, activities, and materials

### Resource Library
1. **Browse Categories**: Resources organized by type and age group
2. **Download Files**: Click download buttons for worksheets, images, documents
3. **Search Resources**: Use search bar to find specific materials

### Blog Section
- **Teaching Tips**: Read articles about effective teaching methods
- **Educational Content**: Access guides and best practices
- **Community Insights**: Learn from experienced teachers

## User Roles

### Editor Role
**Permissions:**
- Create and edit lessons
- Manage blog posts
- Upload resources
- Create teaching tips
- View basic analytics
- Access editor settings

**Restrictions:**
- Cannot manage users
- No system settings access
- No Telegram integration access
- Limited analytics view

### Administrator Role
**Permissions:**
- All Editor permissions
- User management (create/edit other users)
- Full analytics access
- System settings configuration
- Telegram integration management
- Backup management

**Restrictions:**
- Cannot delete Super Admin accounts
- Limited system-level configurations

### Super Administrator Role
**Permissions:**
- All system access
- Complete user management
- System-level settings
- Database management
- Integration configurations
- Full backup and restore capabilities

## Content Management

### Creating Lessons

#### Step 1: Basic Information
1. Navigate to **Lessons > Create New**
2. Fill in required fields:
   - **Title**: Descriptive lesson name
   - **Age Group**: Select appropriate age range
   - **Duration**: Estimated lesson time
   - **Difficulty**: Easy, Medium, Hard

#### Step 2: Content Creation
1. **Lesson Overview**: Brief description and objectives
2. **Scripture Reference**: Biblical passages used
3. **Main Content**: Detailed lesson plan using rich text editor
4. **Activities**: Interactive elements and games
5. **Materials Needed**: List required supplies

#### Step 3: Media and Resources
1. **Featured Image**: Upload lesson thumbnail
2. **Additional Images**: Supporting visual content
3. **Downloadable Resources**: Worksheets, handouts, activities
4. **Video/Audio**: Multimedia content (optional)

#### Step 4: Publishing
1. **Save as Draft**: Work in progress, not visible to public
2. **Publish**: Make lesson live and searchable
3. **Schedule**: Set future publication date

### Managing Blog Posts

#### Creating Posts
1. Go to **Blog > Create New Post**
2. **Title and Content**: Use rich text editor for formatting
3. **Categories**: Organize posts by topic
4. **Tags**: Add searchable keywords
5. **Featured Image**: Upload post thumbnail
6. **SEO Settings**: Meta description and keywords

#### Publishing Options
- **Draft**: Save for later editing
- **Published**: Live on website
- **Scheduled**: Automatic future publication

### Resource Management

#### Uploading Files
1. Navigate to **Resources > Upload New**
2. **File Selection**: Choose documents, images, or media
3. **Categorization**: Assign to appropriate category
4. **Description**: Add helpful details for users
5. **Age Group**: Specify target audience

#### Supported File Types
- **Documents**: PDF, DOC, DOCX, PPT, PPTX
- **Images**: JPG, PNG, GIF, SVG
- **Audio**: MP3, WAV
- **Video**: MP4, AVI, MOV
- **Archives**: ZIP, RAR

### Event Management
1. **Create Events**: Special occasions, holidays, church calendar
2. **Event Details**: Date, time, description, resources
3. **Recurring Events**: Set up repeating occasions
4. **Event Categories**: Organize by type (Holiday, Special Service, etc.)

## System Administration

### User Management (Admin/Super Admin)

#### Creating New Users
1. Go to **Users > Create New User**
2. **Basic Information**:
   - Name and email address
   - Initial password (user must change on first login)
   - Role assignment (Editor, Admin, Super Admin)
3. **Permissions**: Role automatically sets appropriate permissions
4. **Account Status**: Active/Inactive toggle

#### Managing Existing Users
1. **View All Users**: See complete user list with roles and status
2. **Edit Users**: Modify information, change roles, reset passwords
3. **Deactivate Users**: Temporarily disable access without deletion
4. **Activity Monitoring**: Track user login and content creation

### System Settings

#### General Configuration
1. **Site Information**: Name, description, contact details
2. **Email Settings**: SMTP configuration for notifications
3. **File Upload Limits**: Maximum file sizes and allowed types
4. **SEO Settings**: Meta tags and search engine optimization

#### Integration Settings (Admin/Super Admin)
1. **Telegram Bot Configuration**:
   - Bot token from @BotFather
   - Channel ID for content import
   - Webhook URL setup
2. **Newsletter Integration**: Email service provider settings
3. **Analytics Configuration**: Tracking and reporting setup

### Backup Management (Admin/Super Admin)
1. **Automatic Backups**: Schedule regular database and file backups
2. **Manual Backup**: Create immediate backup before major changes
3. **Restore Options**: Recover from previous backup points
4. **Storage Settings**: Local or cloud backup storage configuration

## Troubleshooting

### Common Issues

#### Login Problems
**Issue**: Cannot log in to admin dashboard
**Solutions**:
1. Verify email and password are correct
2. Check if account is active
3. Clear browser cache and cookies
4. Contact administrator for password reset

#### File Upload Issues
**Issue**: Cannot upload files or images
**Solutions**:
1. Check file size (must be under limit)
2. Verify file type is supported
3. Ensure stable internet connection
4. Try different browser or clear cache

#### Content Not Displaying
**Issue**: Lessons or resources not showing on website
**Solutions**:
1. Verify content is published (not draft)
2. Check publication date settings
3. Clear website cache
4. Contact administrator if problem persists

#### Performance Issues
**Issue**: Website loading slowly
**Solutions**:
1. Clear browser cache
2. Check internet connection
3. Try different browser
4. Report to administrator for server optimization
